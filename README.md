# 🔍 CodeSentry — Frontend

React frontend for CodeSentry AI-Powered Java Code Review System.

## 🌐 Live Demo
https://codesentry-ui.vercel.app

## 🔗 Backend Repository
https://github.com/vinayakbhardwajcse/Repository-name-codesentry-backend-Visibility-Public

## 🛠️ Tech Stack
- React 18
- Axios
- CSS3 with Dark Mode

## ✨ Features
- 📝 Paste or upload Java code
- 📁 Drag and drop file upload
- 📊 Circular health score gauge
- 🌙 Dark mode toggle
- 🐙 GitHub repository scanner
- 📄 PDF report download

## 🚀 Getting Started

```bash
npm install
npm start
```
